# PIN Enterprises Website

A modern, responsive static website for PIN Enterprises, premium manufacturers of Hollow and Interlock Bricks.

## Features
- **Responsive Design**: Optimized for all devices including desktops, tablets, and mobile phones.
- **Modern UI**: Clean, professional aesthetic with smooth animations and transitions.
- **Interactive Elements**: Includes a 3D flip-book product gallery and dynamic swipers.
- **Quote Request Form**: Easy-to-use form for clients to request customized project quotes.
- **Integrated Contact Details**: Includes Google Maps integration and direct WhatsApp/Phone call links.

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)
- Swiper.js (Sliders)
- AOS (Animate On Scroll)
- Turn.js (3D Flipbook)
- FontAwesome (Icons)

## Development
To run this project locally, simply serve the directory using any local web server.

Using Python 3:
```bash
python3 -m http.server 3000
```
Then navigate to `https://www.pinenterprises.live` in your web browser.

Using Node.js (npx):
```bash
npx serve -p 3000
```
